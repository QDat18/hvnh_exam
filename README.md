# 🎓 HVNH Exam System

Hệ thống ôn thi trắc nghiệm cho sinh viên Học viện Ngân hàng

## Tech Stack
- Backend: Spring Boot 3.2 + PostgreSQL + Redis
- Frontend: React + TypeScript + Ant Design

## Quick Start
1. `docker-compose up -d` - Start databases
2. `cd backend && ./mvnw spring-boot:run`
3. `cd frontend && npm start`

Chi tiết: Xem `GETTING_STARTED.md`
